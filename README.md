Deskripsi Singkat Sistem

Sistem Pemesanan Makanan Kantin Kampus merupakan aplikasi berbasis Java console yang dirancang untuk membantu proses pemesanan makanan dan minuman di lingkungan kampus. Sistem ini memungkinkan pengguna untuk melihat daftar menu, memilih makanan, menentukan jumlah pesanan, serta menghitung total pembayaran secara otomatis. Aplikasi ini dijalankan melalui terminal dan berfokus pada penerapan logika bisnis serta konsep pemrograman berorientasi objek.

Teknologi yang Digunakan

Teknologi yang digunakan dalam pengembangan sistem ini meliputi:

Bahasa Pemrograman: Java

Jenis Aplikasi: Aplikasi berbasis console

Paradigma Pemrograman: Object-Oriented Programming (OOP)

Tools Pendukung:

Java Development Kit (JDK)

Terminal / Command Prompt

IDE (IntelliJ IDEA, Eclipse, atau Visual Studio Code)

GitHub sebagai media penyimpanan dan distribusi source code

Cara Menjalankan Aplikasi

Aplikasi dijalankan secara lokal melalui terminal dengan langkah-langkah sebagai berikut:

Pastikan Java Development Kit (JDK) telah terinstal di komputer.

Buka terminal atau command prompt.

Masuk ke direktori tempat file program berada.

Lakukan kompilasi program dengan perintah:

javac SistemKantin.java


Jalankan aplikasi dengan perintah:

java SistemKantin


Aplikasi akan menampilkan menu pemesanan pada layar terminal dan siap digunakan.
