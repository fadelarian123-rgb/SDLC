Sistem Pemesanan Makanan Kantin Kampus
Deskripsi Singkat Sistem

Sistem Pemesanan Makanan Kantin Kampus adalah aplikasi yang dirancang untuk membantu proses pemesanan makanan dan minuman di lingkungan kampus secara terkomputerisasi. Sistem ini memungkinkan pelanggan (mahasiswa, dosen, dan staf) untuk melihat daftar menu, melakukan pemesanan, serta melakukan pembayaran secara lebih cepat dan akurat. Di sisi lain, petugas kantin dapat mengelola menu dan memantau pesanan yang masuk sehingga proses pelayanan menjadi lebih efisien.


Teknologi yang Digunakan

Bahasa Pemrograman: Java
Paradigma: Object-Oriented Programming (OOP)
Jenis Aplikasi: Aplikasi berbasis console
Tools Pendukung:
Java Development Kit (JDK)
IDE (IntelliJ IDEA / Eclipse / VS Code)
Git & GitHub (version control dan repository)


Cara Menjalankan Aplikasi

Persiapan Lingkungan
Pastikan Java JDK sudah terinstal (versi 8 atau lebih baru).
Pastikan variabel environment JAVA_HOME sudah dikonfigurasi.

Clone Repository
git clone https://github.com/username/sistem-kantin-kampus.git

Masuk ke Direktori Project
cd sistem-kantin-kampus

Compile Program
javac SistemKantin.java

Jalankan Aplikasi
java SistemKantin


Penggunaan
Pilih menu yang tersedia melalui tampilan console.
Tambahkan pesanan sesuai kebutuhan.
Lakukan pembayaran untuk menyelesaikan transaksi.
